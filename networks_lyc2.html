CSV — cамый простой формат кодирования сетей

  1. Source,Target,Weight
  2. Даня,Маша,2
  3. Вася,Петя,15
  4. Вася,Маша,1

Как можно делать такие CSV?

  1. В блокноте...
  2. В любом табличном редакторе (LibreOffice/OpenOffice/Excel)
  3. Из питона. Например: github.com/DanilSko/lyc_compling/blob/master/Networks_24052018/Text2NetworkSample_Lyceum2405.ipynb
  4. Ручками в нашем Ezlinavis

Gephi (gephi.org)

  • Простой инструмент для анализа сетей (не надо уметь программировать)
  • "like Photoshop but for graph data" (gephi.org)
  • Открытый и бесплатный
  • Написан на Java и работает на всех основных ОС
  • Расширяемый (можно создавать плагины)
  • …немного глючный : )
